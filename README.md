# Workshop-Redux
Learning Redux by Reimplementing it

## Installation

#### Prérequis : 
Installer Yarn :
```
brew update
brew install yarn
```

#### Installation des dépendances :
Via Yarn :
```
yarn install
```

## Développement

Lancer les tests :
```
npm run test
```

Lancer le serveur local :
```
npm start
```
